① 信息提取与处理（4分）
本质：读懂材料，提炼有效信息 不是复制粘贴，要概括、抽象、推理。
读材料三步走
  1. 先看标题 — 确定主角（什么生物/什么地方）
  2. 看图例/坐标轴 — 理解横纵轴含义，找极值、趋势
  3. 圈关键数据 — 最高/最低/转折点 = 答题依据
常见信息提取陷阱
❌ 直接抄材料原话 → 不得分
✅ 用自己语言+学科术语重新表达，说明"说明了什么"
❌ 只看文字，不看图表 → 漏掉关键信息
✅ 文字和图表都要仔细读，信息通常互补
② 问题分析与质疑（7分）⭐核心
分值最高！这里拉开差距 要求用地理 + 生命科学双重视角分析，论证结论。
万能答题链条
地理环境特征（地形/气候/位置） → 对生物产生的影响 → 生物的适应性表现（形态/生理/行为） → 结论
两类高频问法
问"原因/为什么"：先写地理条件，再写生物响应，因果链完整。
例：该地降水少（地）→ 植物根系发达（生）→ 吸收深层水分→ 适应干旱
问"分析/说明"：从两学科各写一点，用①②分点，不要写成流水账。

大白话（-分）学科术语（得分）

③ 结论阐释与创新（4分）
难度最高，但分值固定4分，不要放弃！ 掌握套路就能稳拿3分。
两种题型
A. 解释型："根据上述结论，解释……的原因"
套路：重复材料中的已知结论 → 用"因为……所以……"推导到新现象
B. 方案建议型："请你提出改善建议/保护措施"
套路：从地理角度（工程措施）+ 生物角度（生态措施）+ 管理角度各写一点
万能建议模板
工程措施：建（防护林/水库/梯田/绿化带） 生态措施：退耕还林/建自然保护区/禁止捕猎/减少污染 管理措施：立法/宣传教育/减少人类活动干扰
🔗 食物链 · 食物网（必考）
3条死规则，写错就丢分 ① 起点必须是生产者（植物）；② 箭头方向：被吃 → 吃者；③ 不包含分解者和非生物成分
标准写法示例
草 → 兔 → 狐狸 → 狼
能量流动规律
  • 能量沿食物链流动时，每个营养级只保留约10%～20%
  • 营养级越高，能量越少，个体数量越少
  • 若某种生物数量减少，其天敌数量也会减少（短期）
生态平衡
  • 食物链/网越复杂，生态系统自我调节能力越强，越稳定
  • 人类活动破坏→生物多样性减少→稳定性下降
☀️ 光合作用 · 呼吸作用（核心）
光合作用（白天/光照）
CO₂ + H₂O ──光照+叶绿素──→ 有机物 + O₂ 影响因素：光照强度↑、CO₂浓度↑、温度适宜 → 光合速率↑
呼吸作用（全天）
有机物 + O₂ ──────────→ CO₂ + H₂O + 能量 影响因素：温度越高 → 呼吸越强 → 消耗有机物越多
跨学科黄金公式 有机物积累量 = 光合总量 - 呼吸消耗量
高海拔/昼夜温差大 → 夜间低温 → 呼吸弱 → 积累多 → 口感甜
农业应用
  • 合理密植 → 充分利用光能 → 提高光合效率
  • 中耕松土 → 增加O₂ → 促进根系有氧呼吸 → 利于吸水吸肥
  • 温室大棚增加CO₂ → 促进光合作用 → 提高产量
🐾 生物适应环境（高频考点）
常见适应类型速查
环境特征生物特征原因
干旱（西北/荒漠）根系深长；叶片小/退化成刺；肉质茎储水减少蒸腾，增加吸水
高寒（青藏高原）植株矮小贴地；毛厚/脂肪多（动物）；红细胞多保温御寒；运输O₂适应低氧
强紫外线叶色深绿/发红；皮肤色素多吸收/屏蔽紫外线
水生环境流线型体形；鳃呼吸；侧线感振动减少阻力；水中获O₂
寒冷冬季落叶（减少蒸腾）；冬眠；迁徙降低能耗；躲避低温
♻️ 碳循环 · 物质循环
碳在自然界中的流动路径
大气CO₂ →光合→ 植物有机物 →摄食→ 动物

  • 燃烧化石燃料 → 大量CO₂释放 → 温室效应 → 全球变暖
  • 植树造林 → 固定CO₂ → 减缓温室效应

🌱 生物多样性与保护
濒危原因（必背）
  • 栖息地丧失与破坏（最主要原因）
  • 过度捕猎/采集
  • 环境污染（水/空气/土壤）
  • 外来物种入侵（竞争关系破坏原有生态）
保护措施
  • 就地保护：建立自然保护区（最有效的方法）
  • 迁地保护：动植物园、种质库
  • 法律保护：野生动植物保护法、禁猎禁捕
  • 宣传教育：提高公众保护意识

🌡️ 气候图表判读（必学技能）
读气温曲线降水柱状图（双轴图）4步法
  1. 看纵轴单位：左轴气温（℃），右轴降水（mm）
  2. 看气温曲线：最高/最低月、年较差大小 → 判断大陆性/海洋性
  3. 看降水柱：集中哪几个月 → 雨热同期=季风气候；冬雨=地中海
  4. 综合判断气候类型，描述农业影响
快速判断口诀
夏高温冬寒冷+夏多雨 → 季风气候（温带/亚热带）
全年少雨 → 温带大陆性或沙漠气候
高寒+紫外线强 → 高原山地气候
冬雨夏干 → 地中海气候（考得少但要知道）

🌾 场景模板一：农业分析类
🌾 分析某地农产品品质优良/产量高的原因
地理角度（必答）
①光照充足，太阳辐射强，利于光合作用
②昼夜温差大，夜间气温低，呼吸作用弱，有机物消耗少
③气温较低/海拔高，病虫害少，农药使用少
④水源充足（冰川融水/降雨充沛），灌溉便利
⑤土壤肥沃，矿物质丰富

①光合作用强 → 有机物（糖分/淀粉）积累多
②呼吸作用弱 → 有机物消耗少 → 净积累量高
③生长期长/适温环境 → 充分发育

🐟 2025年真题 · 珍稀水生昆虫与鱼类 深度解析 最新真题
案例背景：研究者在甲（海拔3800m）、乙（海拔2100m）、丙（海拔2000m）三地溪流中，调查珍稀水生昆虫和鱼类的分布情况。乙地距城镇较近，甲地最偏远。表格数据包含各地溪流含氧量、常见昆虫种类数、珍稀昆虫种类数等。

🌟 46题满分答案拆解（最难题，3分）
题目：分析甲乙丙三地中，乙地珍稀水生昆虫种数最少的原因。
  • ①水质污染（1分） 乙地距城镇较近 → 工业/生活污水排放 → 溪流水质变差 → 对污染敏感的珍稀昆虫难以生存，种数减少
  • ②含氧量影响（1分） 污染 + 海拔适中 → 溪流含氧量相对较低 → 水生昆虫生命活动受限 → 珍稀昆虫种数减少
  • ③种间竞争（1分） 乙地常见水生昆虫种类多 → 与珍稀昆虫竞争食物/氧气/栖息地 → 珍稀昆虫受到排挤，种数减少
答题公式：污染（城镇化）→ 环境恶化 → 珍稀物种消失
+ 竞争者增多 → 珍稀物种受压 → 种数减少
⭐ 三点各写一条，分点表达！

🌟 46题满分答案拆解（最难题，3分）
题目：从生物和地理两个角度，分析野生植物G濒危的主要原因。
  • 生物角度①（1分） 传粉昆虫数量减少 → 花粉无法有效传播 → 种子结实率下降 → 种群数量减少
  • 生物角度②（1分） 植物G属于兰科，专性传粉 → 对特定昆虫依赖性强 → 一旦传粉者消失则无法繁殖
  • 地理角度（1分） 人类活动扩张（农业/建设用地）→ 野生植物G栖息地减少/破碎化 → 种群隔离，无法交配，数量下降
答题模板：
生物角度：传粉者减少 → 繁殖受阻 → 种群下降
地理角度：人类活动 → 栖息地减少/破坏 → 无法生存
⭐ 两个角度必须都写，各用一个完整因果链！

🐌 2023年真题 · 陆生蜗牛 深度解析
案例背景：陆生蜗牛主要以植物为食，蜗牛壳由碳酸钙（~95%）组成。研究者通过蜗牛壳中碳元素含量判断当地植被情况，并在全国多个干湿区采集样本。
核心知识链
  • 碳循环 大气CO₂ → 植物光合固定 → 蜗牛取食植物获碳 → 碳酸钙沉积在壳中
  • 干湿分区 湿润区植被丰富 → 蜗牛食物多 → 壳中碳含量高；干旱区相反
  • 研究方法 样方法；随机取样；避免主观选择
  • 跨学科结论 蜗牛壳碳含量 → 反推当地植被 → 判断湿润程度（地理×生物联动）
关键得分句
蜗牛从植物中获取碳元素（1分）
→ 蜗牛壳中碳酸钙越多（1分）
→ 说明该地植被越丰富（1分）
→ 反映当地气候越湿润（1分）
🌿 2022年真题 · 黄芪 关键考点
案例背景：黄芪是多年生草本药用植物，主产于内蒙古、甘肃、山西等地。其药效成分（多糖、皂苷、黄酮）含量与生长环境密切相关。
  • 地理① 内蒙古/甘肃 → 温带大陆性气候 → 干旱少雨，昼夜温差大，光照充足
  • 生物① 根系深长发达 → 吸收深层土壤水分 → 适应干旱
  • 核心联动 昼夜温差大 → 白天光合积累有机物多，夜间呼吸消耗少 → 有机物净积累量大 → 药效成分含量高
  • 举一反三 新疆哈密瓜/棉花/西北水果为什么特别甜 → 同一套逻辑！

💡 3条最重要的提分原则
原则一：材料第一
所有答案的依据来自材料，不是凭空背诵。每道题先找材料证据，再配知识点。
原则二：双学科视角
每个问题都尝试从地理+生命科学两个角度作答，分点写，绝不只写一个学科。
原则三：术语代替白话
把大脑里的"口语直觉"翻译成学科术语再落笔。这一步就是得分与失分的分水岭。
